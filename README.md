# rittai-guriguri-ipad-tw
立体グリグリ「立體繪圖 (Rittai Guri-Guri)」— 繁體中文化版本 —
這是一款專為學習 「第三角法正投影圖」 而設計的 3D 視覺化工具。
本軟體跨越了 30 年的時空，從 MS-DOS 時代進化至 AI 協作開發時代，旨在提供最直觀的空間幾何學習體驗。


📜 歷史與開發背景

原創作者： 川俣純 (Jun Kawamata)

原創專案網址： https://github.com/jkawamata/rittai-guriguri-ipad

2026 年重製版： Vibe Coding & ChatGPT 5.5

於 2026 年 5 月利用現代技術與高階 AI 協作重製，賦予經典工具新的生命。

繁體中文化： 許仁傑 (Jen-Chieh Hsu)


✨ 主要功能
四視圖模式： 同時顯示俯視圖、正視圖與側視圖，完美對應第三角法教學。

STL 支援： 支援匯入外部 STL 3D 模型檔案（建議使用 Tinkercad 等工具製作）。

進階顯示： 支援「隱藏線」顯示、實體表面渲染、以及線條粗細調整。

直覺操作： 支援滑鼠、鍵盤以及遊戲手把操作。

視覺優化： 提供黑白背景反轉功能，適應不同的教學與觀察需求。


🎮 如何使用

載入模型： 點擊「載入」可選擇內建範例或您的 STL 檔案。

觀察結構： 使用「四面模式」觀察正投影圖與立體圖的關聯。

調整細節： 透過選單開啟「顯示隱藏線」，幫助理解物體內部結構。

另存新檔： 編輯後的狀態可以「另存新檔」為專用的 GRI 格式。

注意： 本軟體主要用於模型的「讀取」與「學習觀察」，無法直接「建立」或「匯出」新的 STL 檔案。
